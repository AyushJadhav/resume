Project: Django Application: SnapVibe

SnapVibe allows users to share images, add captions, and interact with posts from other users through likes and comments. The aim of the application is to provide an interface where users can share the photographs of the newly discovered places and suggest such places for a nice visit and also, providing functionalities like uploading, viewing, liking, and commenting on posts of a large community of users.

Technologies Used:
AWS Elastic Beanstalk CodePipeline Django SQLite HTML5 Bootstrap 5 DevOps CI/CD Pylint SonarCloud

The application is deployed on AWS Elastic Beanstalk using CodePipeline for continuous integration and deployment. Pylint performs static code analysis, and security vulnerabilities are managed by SonarCloud. Django, a high-level Python web framework, handles user registrations, authentication, and database interactions using SQLite. HTML5 and Bootstrap 5 are used for the front-end to ensure responsiveness and user-friendly interface.
